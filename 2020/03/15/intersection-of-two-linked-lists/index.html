Leetcode 160 intersection-of-two-linked-lists

发表于 2020-03-15 |

Problem Description

Write a program to find the node at which the intersection of two singly linked lists begins.

Solution

注意本题两个节点相等是指两个node是同一个地址的同一个node，而不是判断值相等的两个node

利用栈

将两个链表都压入栈中，然后出栈，直到节点不同，返回上一个节点。

遍历

取两个指针从头开始遍历链表，当指针为空时指向另一个链表，直到二者均为空或相等。

原理：虽然俩个链表的长度是不同的，但是二者的和是相同的，那么使用两个指针都遍历一次这两个链表，一定会在交叉的地方相遇。
