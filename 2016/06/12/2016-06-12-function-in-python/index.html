Python学习笔记之函数

发表于 2016-06-12 | 分类于 Python |

这篇文章是一边读廖雪峰老师的python教程一边写的有关python的函数特性

1、函数也是对象，因此可以将函数名赋值给一个变量，然后用这个变量调用该函数，相当于为函数起了一个别名。

2、在函数中如果不显式的使用return，也返回结果，结果为None

3、如果函数为一个空函数，则需在函数体内写pass，否则会报错。

4、函数返回多个值实际是返回了一个tuple元组。tuple是一个不可变的对象。如果在函数调用的地方用多个变量接收一个元组，则按位置赋给对应的值。

5、函数的参数可以包含默认参数，默认参数放在必选参数的后面，否则会报错（如果没有这项规定，那在函数调用的时候编译器无法判断传入的值是必选参数还是默认参数）。调用含有默认参数的函数时，可以不按照参数的顺序进行调用，但是必须指明参数的名称。

6、函数的默认参数必须指向一个不可变对象，如果是可变的对象且函数内对该参数进行了修改，那每次调用该函数该参数的值都会发生变化。（关于可变对象与不可变对象的区别，可以查看我的另一篇博客）

7、函数的可变参数是指函数的参数的数量可以动态的改变，这种参数需要在参数名前加一个* ，传入的参数是一个tuple。如果要传入一个tuple那么再要传入的tuple前加*即可。

8、关键字参数允许你传入0个或任意个含参数名的参数，将传入的值组装成dict，参数名前加**（注意与7可变参数的区别）

9、必选参数、默认参数、可变参数、关键字参数这四种参数可以组合使用也可以只用其中的某一些，但是如果用组合参数，那么参数的类型必须为：必选参数、默认参数、可变参数和关键字参数。
