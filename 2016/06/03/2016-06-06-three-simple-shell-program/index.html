三个简单的shell小程序

发表于 2016-06-03 | 分类于 Linux |

创建用户

题目：创建100个用户，前50个在group1中，后50个在group2中，并将user10设为group1的管理员，user51设为group2的管理员

涉及到了for循环和if语句的使用。
有几个要注意的点：

  • for循环的条件要有两个括号
  • for循环使用do表示开始，done表示结束
  • if 语句中如果要用'['和']'的话'['之后和']'之前一定要加空格，否则会报错
  • 用fi来表示if语句的结束

判断文件的类型

题目：判断给定的值是文件还是目录，如果是目录的话显示目录下的文件

涉及到if的分支的书写还有shell脚本中函数的书写
几个要注意的点：

  • 调用脚本时输入的值在脚本中分别用$1 $2 $3....来表示
  • shell中用function来表示一个函数，如果要传递值给函数，只要在调用函数时在之后加上相应的值即可，在函数内部用$1 $2 $3...来表示
