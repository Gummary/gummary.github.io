发表于 2020-08-01 |

01背包问题

问题描述

假设有N个物体，第i个物体的重量为w[i],价值为c[i],背包容量为C,那么将哪些物体放入到背包中，可以在不超过背包容量的情况下使背包所装的价值最大

首先定义状态f[i][v]，其中\(i\in \{1,\dots N\}, v\in \{1, \dots C\}\),则f[i][v]的含义是取前i件物品，容量恰好为v的情况下的最大价值。

则状态转移方程为：

\[ f[i][v] = \text{max}(f[i-1][v], f[i-1][v-w[i]]+c[i]) \]

也即每个物体都有两个状态(01背包的由来)，拿或者不拿。那么：

  • 如果不拿，那么最大收益为取前i-1个物体，容量恰好为v的价值
  • 如果拿，那么最大收益为前i-1个物体，容量恰好为v-w[i]的价值加上当前物体的价值。

注意f[i][v]是有意义的前提是，前i个物体，恰好能组成一个重量为v的背包，因此最终的答案不是f[N][C],因为可能没法恰好组成重量为C的背包，而是\(f[N][0\dots C]\)的最大值。如果移除恰好，那么状态转移方程为：

\[ f[i][v] = \text{max}(f[i-1][v], f[i-1][v-w[i]]+c[i], f[i-1][v-1]) \]

这个算法的时间复杂度为\(O(N*C)\),N为物体数量，C为背包容量，时间复杂度为\(O(N*C)\)，观察这个可以发现， 每次判断时仅使用了dp数组每一行的上一行，因此可以将空间复杂度降低到\(O(C)\)

注意内层循环必须是从最大的背包容量开始迭代，这样可以保证推导dp[v]中使用的dp[v-w[i]]是dp[i-1][v-w[i]]的状态。

例题

leetcode 416. Partition Equal Subset Sum

题目及分析

给定n个数，问能否找到其中一个子集，使子集的和为n个数和的一半。

每个数有两个状态，加入子集或不加入子集，最终使加入子集的元素的和恰好为全部和的一半，因此可转换为01背包问题，数为物体的重量，最终使背包内物体的重量恰好为target.

多重背包问题

问题描述

给定N个物品，第i件物品的重量为\(w[i]\),价值为\(v[i]\),数量为\(n[i]\),背包容量为C，在不超过背包容量的情况下取得最大的价值。

题目分析

本题与完全背包类似，但是每个物品的数量有限制，与完全背包一样，可以将每个物品拆分成n[i]个，将该问题转为01背包问题，此时时间复杂度为\(O(NV\sum n_i)\)，状态转移方程为：

\[ f[i][v] = \text{max}(f[i-1][v-k\times w_i),\quad 0 \leq k \leq n_i \]

另一种做法是将每个物品的数量用二进制表示，也即每个\(n_i\)可表示成\(1,2, 4, \dots, 2^x, n_i-(2^{x+1}-1)\).其中，\(1,\dots,2^x\)可表示\([1,2^{x+1}-1]\)内的所有数，而\(n_i-(2^{x+1}-1)\) 与\(1,\dots,2^x\)可表示\(2^{x+1},n_i\)之间的所有数。此时，算法时间复杂度为\(O(NV\sum log(n_i))\)

二维费用背包问题

问题描述

对于每个物品，具有两个不同的代价，拿取某个物品必须同时支付这两个代价，这里的代价可以理解为每个物品的重量，每个物品的代价为\(a[i],b[i]\)可支配的两种重量总价为\(V,U\).

问题分析

代价多了一维，那么状态也增加一维即可。此时，状态转移方程为\(f[i][v][u] = \text{max}(f[i][v][u],f[i][v-a[i]][u-b[i]])\),与01背包类似，也可以仅使用二维数组表示状态，当为完全背包时，逆序遍历；当为多重背包时，将物品分组。

二维费用的背包问题还可以另一种形式出现，背包的容量为V，但是只能取M个物品，此时，也是二维费用的背包问题，每个物品的第二维代价均为1，容量为M。

依赖背包问题

问题描述

如果要拿物品i那么必须拿物品j，称这种问题为有依赖的背包问题。为了简化问题，假设存在不依赖其他物品但被其他物品依赖的物品，并且每个有依赖的物品仅会依赖一个物品。

我们将不依赖其他物品的物品称作主件，依赖其他物品的称作附件，那么可以根据主件和附件的关系将所有物品划分为多个组。那么情况变为要么不拿某个主件，要么只拿某个主件，要么拿某个主件并拿一个附件，要么拿一个主件并拿两个附件，依次类推。所以可以将所有物品划分成不同的组，但是每组内的物品可以随意选择，每个组内共有\(2^k+1\)中组合方式。而对于每个组内的每个附件的状态是要么拿要么不拿，很像01背包问题，因此我们可以先在组内做一次01背包，得到\(C-W_k+1\)个代价，其中\(W_k\)为主件k的代价，费用为c的物品的价值为\(f[c-W_k]+V_k\).这些代价之间是由该组内的物品通过不同组合方式组成的，因此彼此之间是互斥的，此时可以直接应用分组背包问题的解决思路解决。
