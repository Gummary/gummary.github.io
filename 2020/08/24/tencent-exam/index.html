队伍方案数量

某部门有n个人，从n个人中选出任意数量的人组成队伍，从这些人中选出一个当作队长，求方案数对\(10^9+7\)取余的结果。

数学公式推导为：

\[ \text{ans} = C_n^0\times 0 + C_n^1\times1+C_n^2\times2+\dots+C_n^n\times n \]

将其反写有：

\[ \text{ans} = C_n^n\times n+C_n^{n-1}\times{n-1}+\dots+C_n^0\times 0 \]

根据性质\(C_n^r = C_n^{n-r}\), 将两个ans相加后得到：

\[ 2\text{ans} = n\sum_{i=0}^nC_n^i = n2^n \]

其中\(\sum_{i=0}^nC_n^i=2^n\)为二项式定理

则：

\[ \text{ans} = n2^{n-1} \]

最短路径问题

现在有n个点，有m条双向路径，有k个传送门，传送门表示两个路径之间的距离为0，传送门为单向的，求从0到n之间的最短路径。

利用Dijkstra算法求最短路即可。

Dijkstra算法是，假设求P点到其他所有点之间的最短路径，则维护一个数组dist，值为p点到每个点的最短距离；然后维护一个set，内容为已经求出最短距离的点集。则具体步骤为：

  1. 初始化dist数组，dist[p] = 0，将所有与p相邻的点加入到一个队列中
  2. 从队列中选出与p距离最小的点q加入最短点集中，将与q相邻的点加入到队列里，更新所有不在最短点集中的点的距离
  3. 重复2，直到队列为0
