geetest验证码破解

发表于 2017-01-21 | 分类于 Python |

晚上的时候想做一个论坛自动注册发帖的Python爬虫，结果注册的时候出现了geetes的滑动验证码，人工注册一个账号之后在登陆的时候这个滑动验证码又出现了，然后就找了一下用python破解该验证码的方法，发现还真有，但是是用Python2.7写的，而我的开发环境是Python3.6，所以就将他的代码移植到了Python3上。

参考文章的地址为: 破解极验(geetest)验证码

一 准备工作

1 实现鼠标的拖拽移动，这里使用的是Selenium，chromedriver

根据自己的chrome版本到下面的地址下载chromedriver

ChromeDriver

2 复原图片

通过http请求获取到的破损的图片和完整的图片均为乱的，并不是用户看到的，是通过CSS组合起来的，所以还需要将乱的图片保存下来，将其改为正确的图片。这里使用的python库是requests(获取图片),Pillow(恢复图片)

3 生成轨迹

生成轨迹的方法直接使用了参考博文的方法，先通过比较RGB的值找到缺口的位置，再随机的生成每次移动的距离。

二 实现思路

1 获取图片

geetest加载验证图片的方法共有4种，鼠标悬浮在轨迹球上时加载、直接加载、鼠标点击后加载、移动端弹出，参考博文实现了直接加载图片方式的破解，我这里讨论第一种。所以在获取图片之前首先要完成鼠标悬浮的动作，然后用requests发送get请求获取图片。

2 恢复图片

在加载完图片之后，HTML代码中会有各个图片正确的位置，所以可以通过正则表达式匹配出每个图片的实际位置，然后使用Pillow实现变换即可恢复正常的图片

3 计算缺口位置

比较两幅图片每个像素点的值，如果相差超过50则为缺口的位置

4 生成轨迹

由于geetest对拖动时的轨迹进行判断，识别是否是人拖动的，所以还需要模拟一下人的行为

四 其他

  1. 如果直接使用原博的代码，我这里仍会出现验证码被怪兽吃掉了的情况，所以在最后改了一下移动的间隔时间。
  2. 验证码验证完成后应该是得到一个geetest_challenge的值，我这里直接用js写到了html中，所以直接获取即可。
