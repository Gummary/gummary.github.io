Maximal-Square

Posted on 2020-03-10

Problem Description

Given a 2D binary matrix filled with 0's and 1's, find the largest square containing only 1's and return its area.

Example:

Solution

使用动态规划算法，对于一个点来说，如果构成一个正方形，则只有两种情况：

  • 自己构成一个面积为1的矩形
  • 与左、左上、右三个点构成更大的矩形

然后用动态规划算出该点所在正方形的边长，对应上述两种情况：

  • 边长为1
  • 左、左上、右三个点构成的最小正方形的边长+1，因为当前点指定使矩形的边长拓宽1

假设用C[i,j]保存边长，M[i,j]为矩阵则状态转移方程为：

\[ C[i,j] = \begin{cases} 0,\quad M[i,j] = 0 \\ Min(C[i-1,j], C[i-1,j], C[i-1,j]) \end{cases} \]

k-th-smallest-prime-fraction

Posted on 2020-03-09

Problem Description

A sorted list A contains 1, plus some number of primes. Then, for every p < q in the list, we consider the fraction p/q.

What is the K-th smallest fraction considered? Return your answer as an array of ints, where answer[0] = p and answer[1] = q.

Note:

  • A will have length between 2 and 2000.
  • Each A[i] will be between 1 and 30000.
  • K will be between 1 and A.length * (A.length - 1) / 2.

Solution

最小堆

将最小值加入最小堆中，从最小堆中取出最小的值，增大这个值的分子和分母，获得两个比这个值大的两个分数，如果之前没有遇到这个分数，则将该分数加入最小堆中，不断重复上述过程，直到取出的值的数量为K。

二分查找

因为除法的值一定小于1所以可以将边界设置为[0,1]在这个范围内进行二分查找，设置中间值为0.5，计算所有分数中小于这个中间值的个数，不断进行二分查找直到找到一个值使得小于mid的分数等于K，每次记录小于这个值的分数的最大值。
