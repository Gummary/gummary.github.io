符号数据

首先利用愿望思维构造出了一些判断表达式，这些表达式构造出了一层抽象屏蔽，利用这些表达式就可以根据求导的公式写出求导的过程。

练习2.64

这题也没做出来，还是找了篇题解翻译了一下Orz

原文

a)PARTIAL-TREE把一个链表分成了三部分：中间部分为THIS-ENTRY，比中间小的，还有比中间大的。然后他创建了一颗二叉树，根节点为THIS-ENTRY，左子树为比中间小的，右子树为比中间大的。

b)将有序表分成两部分分别是由(QUOTIENT (- N 1) 2) 和 (- N (+ LEFT-SIZE 1))来完成的，这两个过程都花费的是常量级的时间，链接左右子树的过程是由 (MAKE-TREE THIS-ENTRY LEFT-TREE RIGHT-TREE)完成的，这个也是常量级的时间，因此，花费的时间为

T(n) = 2T(n/2) + Θ(1)

根据主定理，我们有a = 2, b = 2, f(n) = Θ(1). 因此, T(n) = Θ(n).

list->tree所用的时间为partial-tree和length所用的时间之和，二者的时间均为O(n)，所以list->tree所用的时间为O(n)
